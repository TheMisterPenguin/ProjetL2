◆ deplacement_x_entite()

bool deplacement_x_entite ( t_map *  m,
t_aff *  texture,
int  x,
SDL_Rect *  r 
)
Auteur
Ange Despert

Cette fonction permet à une entité de se déplacer sur l'axe x.
Cette fonction gère les collisions et empêchera l'entité de sortir des limites de la map.
Cette dernière prend également les collisions définies dans la liste des collisions de la map : liste_collisions.
Il est a noté que pour éviter les déplacement trop rapides la fonction utilise l'entier duree_frame_anim qui permet d'êmpécher le déplacement tout les x frames.

Paramètres
mLa map actuelle
textureLa texture de l'entité que l'on veut bouger
yLa nouvelle coordonnée du rectangle de l'entité
rLe rectangle représentant la zone de collision de l'entité
Renvoie
vrai : Si l'entité a réussi à se déplacer
faux : Si l'entité n'a pas pu se déplacer
