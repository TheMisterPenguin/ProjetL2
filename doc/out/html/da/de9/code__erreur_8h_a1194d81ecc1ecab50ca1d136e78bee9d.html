enum types_erreur

L'énumération des codes d'erreur du programme.

Cela permet une gestion plus simple des erreurs. L'utilisateur peut donc connaitre aisaiment le code d'erreur et le message correspondant. Permettant au developpeur de répondre à l'erreur en question.
