◆ type_sort_t

enum type_sort_t

Cela permet de différencier les sorts pour réaliser des actions particulières en fonction de ce type.

Valeurs énumérées
SP_WITCHER 

sort du Witcher

SP_BOSS_BALL 

le sort "boule" du boss

SP_BOSS_GROUP 

le sort "groupé" du boss

SP_BOSS_SPLIT 

le sort "split" qui est créé à partir du sort "groupé" du boss
