◆ action_monstre_t

enum action_monstre_t

Cela permet de savoir dans quelle état est le monstre

Valeurs énumérées
MONSTRE_MARCHER 

le monstre marche normalement

MONSTRE_EN_GARDE 

le monstre est immobile

MONSTRE_ATTAQUE 

le monstre attaque

RUSH_OU_FUITE 

le monstre va vers le joueur ou le fuit

MONSTRE_BLESSE 

le monstre prend des dégats

MONSTRE_PAUSE 

le monstre se stop (après avoir bléssé un joueur)
